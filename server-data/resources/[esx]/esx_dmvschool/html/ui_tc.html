歡迎來到駕訓班


所有實驗市市民開車之前，都必須通過此項考試。
花點時間，用常識回答，不要隨意回答。

筆試
- 理論考試費用為200元，如果考試不及格，則不會退還。
- 不要害怕，駕駛學校會接受付款，但要小心不要背負債務。
- 如果您第一次考試不及格，您將無法立即參加考試，以後才能參加考試。

路考
- 駕駛考試的費用為500元，就像筆試一樣，如果您不及格，這筆款項將不予退還。
- 確保您在開車時保持警惕，避免發生意外!
